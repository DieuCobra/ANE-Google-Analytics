createProduct()method
public function createProduct(sku:String, name:String, price:Number, quantity:uint):IProductBuilder

Create a new Product builder.

The builder returned can be used to setup the optional parameters of the product and then add it to the current transaction.

Parameters

sku:String — the product SKU which uniquely identifies it. The sku cannot be null or the empty string.
 
name:String — the name of the product. The name cannot be null or the empty string.
 
price:Number — the price of the product, expressed in your currency (will be translated in microns automatically).
 
quantity:uint — the quantity purchased.

Returns
IProductBuilder — a new Product builder to finalize the product creation
