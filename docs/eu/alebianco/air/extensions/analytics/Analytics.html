Classpublic final class Analytics
InheritanceAnalytics Object
Implements IAnalytics

The entry point of the extension.

The class is implemented as a Singleton and will manage the trackers and provide facilities to dispatch the collected hits and set behavioral flags.

dispatch()method 
public function dispatch():void

Dispatches queued hits (view, events, etc) to Google Analytics if a network connection is available.

This has no effect on the dispatch interval and it can be used in conjunction with automatic dispatching to force the dispatch at any point, like before exiting the application.

dispose()method 
public function dispose():void

Dispose the extension and clear any object reference to ready it for garbage collection.

All existing trackers are closed immediately and can't be used anymore.

Existing references to the class will be unusable.

The extension can be initialized again by calling getInstance()

getTracker()method 
public function getTracker(trackingId:String):ITracker

Returns a tracker for the specified Google Analytics account ID.

When the tracker is created, some information about the current application (id, name and version) are automatically set. They can be overridden before tracking any kind of data.

Multiple trackers can be created for multiple accounts. If a tracker for a given ID has already been created, and not yet disposed, the existing instance will be returned.

Parameters

trackingId:String — String in the form UA-xxxx-y

Returns
ITracker — The tracker from which send tracking events to Google Analytics
